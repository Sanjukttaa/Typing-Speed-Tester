# Typing Speed Tester

A Java Swing desktop application that helps users improve their typing speed and accuracy through interactive typing tests.

## Features

- Four difficulty levels
  - Easy
  - Medium
  - Hard
  - Coding

- Test durations
  - 15 seconds
  - 30 seconds
  - 60 seconds

- Random passage generation
- Live WPM calculation
- Accuracy calculation
- Error counting
- Personal best tracking
- Test history
- Anti-paste protection
- Clean Java Swing interface

## Technologies Used

- Java
- Java Swing
- File I/O
- Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Compile `Main.java`.
4. Run the application.
