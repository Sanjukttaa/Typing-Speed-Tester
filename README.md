# Typing Speed Tester

A Java Swing desktop application that measures typing speed, accuracy, and words per minute (WPM). The application provides multiple difficulty levels, tracks personal best scores, stores typing history, and includes basic anti-cheating features for a fair typing experience.

---

## Features

* Four difficulty levels:

  * Easy
  * Medium
  * Hard
  * Coding

* Three test durations:

  * 15 seconds
  * 30 seconds
  * 60 seconds

* Random passage generation

* Live WPM calculation

* Accuracy calculation

* Error counting

* Personal Best tracking

* Typing test history

* Anti-paste protection

* Automatic test completion when the passage is finished

* Responsive Java Swing user interface

## How to Run

1. Clone the repository.
2. Open the project in VS Code or IntelliJ IDEA.
3. Compile and run `Main.java`.
4. Select a difficulty level and test duration.
5. Start typing and view your results.
